# AI-Powered Regulatory Compliance Checker for Contracts

## Project Overview
An advanced AI system that automates contract review and ensures compliance with regulatory standards (GDPR, HIPAA) using Large Language Models.

## Features
- **Contract Upload**: Support for PDF and DOCX files
- **Clause Extraction**: AI-powered identification of key clause types
- **Risk Assessment**: GDPR and HIPAA compliance checking
- **Interactive Dashboard**: Real-time analysis results

## Tech Stack
**Backend:**
- Python 3.9+
- Flask (REST API)
- OpenAI GPT-4
- PyPDF2

**Frontend:**
- HTML/JavaScript (Simple interface)
